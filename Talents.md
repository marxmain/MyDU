### ImprovedCoreUnitExpertise
```yaml
ImprovedCoreUnitExpertise:
  parent: Talent
  displayName: Improved Core Unit expertise
  group: Architect
  description: Large Core Unit unlock
  order: 6
  requiredTalents: []
  costs: [1000000, 2000000, 3000000, 4000000, 5000000]
  effects: []
```
There can only ever be one required Talent.
